or Operator Evaluates the expression as TRUE if at least one of the expression returns TRUE. Otherwise, if both expressions read FALSE, the entire expression returns FALSE.
True or False #Evaluates to True
True or True #Evaluates to False
False or False #Evaluates to False
1 < 2 or 3 < 1 # Evaluates to True
3 < 1 or 1 > 6 # Evaluates to False
1 == 1 or 1 < 2 #Evaluates to True
Equal Operator == Used to determine if the values are the same. If they are, the expression returns TRUE; if not, the expression returns FALSE.
Not Equals Operator != Used to determine if both vaules are the NOT the same. If they are the same, then the result is FALSE. If not, the result is TRUE.
Comparison Operators Relational operators compare two values or expressions. The return value is either TRUE or FALSE.
if Statement Used to determine the excution of code based on the evaluation of a Boolena expression.
else Statement Code to be run if the expression in the IF statement returns FALSE.
and Operator Compares to sides; if both sides result in TRUE, than the entire expression returns TRUE. If one or both sides result FALSE, than the entire expression returns FALSE.
Boolean Values Booleans are a data type which have two values: TRUE and FALSE.
not Operator Used to evaluate the expression to its inverse(opposite) value. If the value is TRUE, the not operator will return FALSE, and vice versa.
