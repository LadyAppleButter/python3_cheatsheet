Variables Used to store data. The data can be a number, Boolean, string, or another data type. Every variable has a name which can contain numbers, letters and an underscore (-). The equal sign (=) is used to assign a value to the variable.
food_type = "apple banana orange apple"
num_apple = 2
veggie = False
Integers A number is written without a fraction or decimal point. Can be negative, positive or 0.
bunch_bananas = 5
bag_apples = 10
slice_pizza = 8
Floating Point Name Number than contains a decimal point.
pi = 3.14159
meal_cost = 9.99
tip_percent = 0.15
Arithmetic Operators that can be performed on numbers, variables, or some combination of the two. These operators include:

+ for addition
- for subtraction
/ for division
* for multiplication
% for modulus (return the remainder)
** for exponent

add_num = a + z
sub_num = a - z
div_num = a / z
mul_num = a * z
mod_num = a % z
exp_num = a ** z
Plus-Equals Operators Adds values to an existing variable or ,if the variable contains strings, adds the string to the existing string (concatenation).
# Numbers - result will be 10
counter = 5
counter += 5
#Strings - result will be "Apples Oranges"
fruit = "Apples "
fruit += "Oranges"
Modulo Operator Caculates the return the remainder of a division between the first and second number
#Result will return 0
zero = 10 % 5
#Result will return 2
nonzero = 10 % 8
String Sequence of chaaracters (letters, numbers, whitespace or punctuation) enclosed by (either double or single) quotations marks.
fruit = "apple"
fruit_type = "Red Delicous"
String Concatenation Joining strings together using the + operator. Variables and strings can be joined together; joining variables requires the str() function. White space matters when concatenating.
#Prints "Fruit Salad"
first = "Fruit "
second = "Salad"
result = first + second
