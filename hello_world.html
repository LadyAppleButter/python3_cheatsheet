Arithmetic Operators that can be performed on numbers, variables, or some combination of the two. These operators include:

+ for addition
- for subtraction
/ for division
* for multiplication
% for modulus (return the remainder)
** for exponent

add_num = a + z
sub_num = a - z
div_num = a / z
mul_num = a * z
mod_num = a % z
exp_num = a ** z
